# Quick Start

Double-click `launch.bat` to start the Options Builder dashboard.

**Requirements:**
- Conda environment `options-tool` must be set up
- Bloomberg Terminal must be running
- Python 3.12+ with all dependencies installed

**Manual start (if needed):**
```
conda activate options-tool
python -m frontend_dash.run_vnext
```
Then open http://127.0.0.1:8051/v2 in your browser.
